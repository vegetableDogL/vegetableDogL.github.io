1、html（超文本标记语言）基础

1、html的基本结构

<!doctype html><!--指定当前的文件为html文件格式-->
<html><!--html根标签，所有的内容都写在这个标签下-->
    
    <head><!--标签内的内容不用于显示，作用：用于定义当前页面的一些元数据信息和引入第三方的css js 等文件-->
        <meta charset="utf-8"><!--定义当前文件的编码格式-->
        <link href="xxx.css" rel="stylesheet"><!--明天会学：引入外部的css资源-->
        <script src="xx.js"></script><!--明天会学：引入外外部的js资源-->
        <title>我的网页</title> <!-- 标题-->
    </head>
    
    <body><!--body中的内容用于浏览器显示的-->
        <h1>标签1</h1>
        <p>标签1</p>
        <h3>标签1</h3>
    </body>

</html>

6、表单

作用：收集用户填写的数据信息,将收集到的数据提交给指定的服务器进行保存

6.1 form属性

form的常用属性

常用属性 作用
action 将数据指定给服务器的地址
method 提交数据的方式，有两种方式：GET或POST

get和post的区别

提交方法 特点
GET 默认是get方法提交，表单提交的所有的数据显示在地址栏上 地址与参数之间使用?分隔，多个参数之间使用&分隔
POST 数据提交更加安全，不会显示在地址栏上 
form 标签是表单标签
    action 属性设置提交的服务器地址 
    method 属性设置提交的方式 GET(默认值)或 POST 表单提交的时候，数据没有发送给服务器的三种情况： 
        1、表单项没有 name 属性值 
        2、单选、复选（下拉列表中的 option 标签）都需要添加 value 属性，以便发送给服务器 
        3、表单项不在提交的 form 标签中 

GET 请求的特点是： 
    1、浏览器地址栏中的地址是：action 属性[+?+请求参数] 
        请求参数的格式是：name=value&name=value
    2、不安全 
    3、它有数据长度的限制 

POST 请求的特点是： 
    1、浏览器地址栏中只有 action 属性值 
    2、相对于 GET 请求要安全 
    3、理论上没有数据长度的限制

6.2 表单控件

常用表单控件

表单项 控件代码 属性 备注
文本框 type：输入控件的类型
name: 提交参数给服务器时使用的名字
value: 文本框中的默认值
readonly: 内容是只读的，不能修改
disabled: 不可用，灰色显示，数据不能提交给服务器
placeholder: 输入前数据前的提示信息		 如果没有指定type类型，默认是text
密码框 属性同上 输入的字符不可见
单选框 checked：默认选中这项 value: 选中这一项的值，如果没有指定，默认值是on 名字相同的单选框是同一组，一组中只能选中一个。
复选框 checked：默认选中这项 value: 选中这一项的值，如果没有指定，默认值是on
下拉列表 name: 给服务器使用的名字 multiple：多选 size: 多选时显示几项
<option> 代表其中一项 value: 选中这一项的值，如果没指定，值为外面的文本 selected: 默认选中这项
隐藏表单域 name: 给服务器使用的名字 value: 隐藏域的值 表单上不可见，可以将数据提交给服务器。一般用于一些不需要用户看到的数据。
文件域 accept: 指定选择的文件类型 如：image/*
多行文本域 rows:显示的行数 cols:显示的列数 注：没有value值，主体部分就是它的值
提交按钮 <input type=”submit” value=”注册”/> value: 按钮上出现的文字 button也可以指定type类型，默认是提交按钮 将整个表单提交给服务器
重置按钮 还原到没有输入数据前的状态
普通按钮 在表单中没有具体的功能，主要用于后期程序开发
图片按钮 x,y: 鼠标点击在图片上的坐标 具有与submit相同的功能

总结：

1.回顾常用标签

1.字体标签
    标题标签：h1~h6 字体大小逐次递减
        属性：align:3种对齐方式：left(默认)、center、right
    水平线标签：hr
        属性：    
            size:水平线的高度或者厚度
            width:宽度
            color:颜色
            align：对齐方式，默认居中对齐
    font字体标签：过时的标签，可以对字体进行处理
        属性：
            color:字体颜色
            size:字体大小，从1到7逐次递增
    字体加粗标签：b或者strong标签
    字体倾斜：i标签
    段落标签：p标签，作用：1.自动换行 2.段落与段落之间留白；
    换行标签：br标签
2.图片标签
    img标签
        属性：
            src:引入外部图片资源，有2种方式
                    方式1：使用相对路径引入图片：./ 或者../
                    方式2：使用绝对路径：file:///D：aa.jpg
            width/height:指定图片的宽和高
            alt:当图片资源加载不到时，友好提醒；
            title:当鼠标悬停到图片上时，提示信息
3.html列表标签
    有2种形式的列表
        1.有序列表
            <ol type="指定排序的序列号：1，a,A,i,I" start="排序序号的起始位">
                <li>001</li>
                <li>001</li>
            </ol>
        2.无序列表
            <ul type="指定无需列表的排序符号：circle-空心圆，disc-实心圆，square-方格">
                <li>001</li>
                <li>001</li>        
            </ul>
4.超链接标签
    语法格式：<a href="url地址" target="_blank">点击一下</a>
    作用：1.资源跳转 2.使用href="#"完成类似页面刷新的操作 3.锚点定位
    属性：
        href：
            1.具体url：访问具体资源
            2.#：完成类似页面刷新的操作
            3.href="#某个标签的id"，锚点定位；
        target：
            _self:资源跳转时覆盖旧的页面（默认）
            _blank:资源跳转时，不会覆盖旧的页面，会在浏览器开启一个信息的窗口；
5.表格标签
    语法格式：
        <table>
            <tr><th>编号</th><th>姓名</th><</tr>
            <tr><td>001</td><td>张三</td></tr>    
        </table>
    说明：
        table标签：
            属性：border,bgcolor,width,height,align,cellpadding(表格内容距离表格的边框),cellspacing（表格边框与边框之间举例）
            子标签：
                tr:表示行标签
                    属性：align,bgcolor(就近原则)
                    子标签：td(默认居左，则字体不加粗)/th（默认居中，且字体加粗）
                        属性：rowspan(行级合并)、colspan(列级合并)

2.form表单标签

作用：收集客户端输入的数据，并发送给服务器；

语法格式：

<form action="form表单提交的url地址" method="form提交的方式：get/post">
    ........
    <input type="submit" value="提交"/>
</form>

2.1 input输入类型的标签

1.文本输入框
     <input type="text" name="xxxx"/>
2.密码输入框
    <input type="password" name="xxxx"/>
3.单选框
    单选框中，单选项的name属性对应的值要一致，这样指定所有单选项属于共同的一组；
    <input type="radio" name="ra" value="aa" checked/><input type="radio" name="ra" value="bb"/>
4.多选框
    多选框中，多选项的name属性对应的值要一致，这样指定所有选项属于共同的一组；
    <input type="checkbox" name="che" value="bb" checked/><input type="checkbox" name="che" value="cc"/>
5.文件输入框
    <intput type="file" name="xxxx"/>
6.文本域
    作用：输入大文本内容，比如：文章评论、留言等；
    <textarea name="xxx" rows="高" cols="宽度"></textarea>
7.隐藏输入框
    特点：1.客户端看不到效果 2.可以form表单提交
    <input type="hidden" name="xxx"/>

2.2 下拉选

应用场景：1.省市县3级联动
语法格式
    <select name="xxx">
        <option>001</option>    
        <option selected value="02">002</option>  
    </select>
说明：option标签如果没有设置value属性，则提交的是opion标签包裹的内容，否则，提交的是value对应的值；

2.3 按钮

常用三类：
    普通按钮：<input type="button" value="button的名称"> <button>button名称</button>
    form表单重置按钮：<input type="reset" value="表单重置"/>
    什么是表单重置？清空表单，回归原来的状态；
    form表单提交：<input type="submit" value="表单提交"/>

2.4 h5特性

1.输入框提醒
<input type="text" name="用户名" placeholder="请输入一个用户名称"/>
2.日志
<input type="date" name="birthday"/>
3.数值类型
<input type="number" max="120" min="0" name="age"/>
4.邮箱
<input type="email" name="xxx"/>
总之，date,number,color,email等类型的输入框，在进行form表单提交时，如果不符合规范，表单不能提交；
