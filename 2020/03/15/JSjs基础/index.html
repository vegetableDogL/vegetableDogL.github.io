1、JS基础

1、js的组成

组成部分 作用
ECMA Script 构成了JS核心的语法基础(简称es)es6
BOM Browser Object Model 浏览器对象模型，用来操作浏览器上的对象
window、navigator 、screen 、history 、location 、document、 event等；
DOM Document Object Model 文档对象模型，用来操作网页中的元素（标签）

2、js引入的2种方式

1. 内部引入-直接在当前页面中引入js代码
    示例：<script>
            console.log("js控制台")
        </script>
2. 外部引入-引入外部js文件
    示例：<script src="../js/hello.js"></script>

注意：一般css放在head里，js放在body后面（html是从上往下加载的）

3、js的3种输出方式

三种输出方式:
1. 在网页上输出 (相当于网页上写内容)
        document.write("网页上输入")    可以加载标签
2. 控制台输出(一般用户debug js代码)❤❤❤❤
        console.log("控制台打印日志");//类似java中的sout
        快捷方式: 内容.log
3. 弹框❤❤
        alert("弹框输出")
        会遮罩当前网页;

4、js的变量和数据类型

【1】原始类型
    1. number(数值类型，无论是整数还是小数，都属于number类型)
    2. string (字符串,推荐使用单引好表示,但是双引号也可)
    3. boolean (布尔类型，true/false)
    4. undefined (只声明变量,不赋值, 即为未定义)
    null（值） (空,对象为null,undifined派生自null,console.log(null == undefined)==>true)

【2】引用类型(对象)
    所有对象都是继承object
    
如何获取一个变量的类型?
    let type = typeof 变量名称;

js用let声明变量，用var会出现变量的溢出问题（超出作用域范围）。

5、基础语法

基本和JAVA相同

1、运算符

js运算符跟java不同之处:
1.+符号
    1.1 任何类型+string结果都是string(此时的加号就是字符串拼接)
    1.2 其余任何都能运算
            boolean: true=1, false=0
            null : 0
            undefined : NaN (not a number)
2.==(===恒等于)
==  只比较字面值,不用管类型 (常用)
=== 不仅比较字面值,还要比类型 (不常用)

2、流程控制

关注if判断和分支与java的不同之处：❤❤❤

js中的任何类型,在判断条件中,自动转换成boolean
    1. number(非0为true, 0为false) 
    2. string(非空串为true, 空串为false(包括‘’空字符串和undified))
    3. null(等价于false)
    4. undefined(等价于false)

3、循环控制

2. 增强for循环
        for(let element of array){
            //element表示被遍历对象/集合的每个元素
            需要执行的代码;
        }
        
3. 索引for循环
        for(let index in array){
            //index表示 对象/数组的索引
            需要执行的代码;
        }



let array = ['a','b','c']; //js数组符号是中括号,不是大括号

for(let element of array){
    console.log(element); //打印出来的是值
}

for(let index in array){
    console.log(index); //打印出来的是索引
}

// js对象
let obj = {
    name : "张三",
    age : 18,
    married : true
}
// console.log(obj.name);
for(let index in obj){
    console.log(index);//打印出来的是对象的key
    console.log(obj[index]);//打印的是值
    console.log(obj.name);//打印的是是值
}

6、js函数

1、普通函数

function  函数名(参数列表){
    函数体;
    [return 返回值;] // 中括号意思表示内容可以省略....
}

注意：

1. 如果有返回值,直接return;
2. 参数之前不用加任何关键字;
3. 调用函数时, 实参个数不必跟形参个数一致;
4. 函数内部有一个隐式变量 arguments(arguments是数组, 会接收所有传入的参数)
5. js函数没有重载机制;

例子：

 function methord2(name,age) {
     console.log("name-age:",name,age)
     //内置的对象：arguments,可以获取出入函数的所有值
     for (let data of arguments){
         //获取动态出入的所有参数
         console.log(data);
     }
 }
 //name-age: zhagnsan 18
 //methord2("zhagnsan",18);
 methord2("lisi",20,"北京");

2、匿名函数

匿名函数通常与事件结合使用

  1. 命名函数
      function 函数名(参数列表){
          函数体
      }
  2. 匿名函数
      变量 = function (参数列表){
          函数体
      }

  匿名函数注意点:
      1. 在js中,函数也是一种特殊的变量类型
      2. 匿名函数必须给变量

    匿名函数的运用场景:
      1. 对象定义
      2. 事件注册


    let obj = {
        name : "张三",
        age : 18,
        speak : function (x) { // 匿名函数写法
            console.log(this.name + "说" + x);
        },
        eat(){ // 命名写法
            console.log(this.name + "正在吃饭");
        }
    }
    obj.speak('今天天气真好')
    obj.eat()



    let method02 = function () {
        console.log("匿名函数");
    }
    method02() // 通过变量名调用

7、js事件

1. 点击事件：
        1. onclick：单击事件
        2. ondblclick：双击事件 (double: 时间间隔很短两次单击)
        
2. 焦点事件
        1. onblur：失去焦点(绑定鼠标离焦事件)
        2. onfocus:元素获得焦点。(绑定鼠标聚焦事件)

3. 加载事件：
        1. onload：页面加载完成后立即发生。

4. 鼠标事件：
        1. onmousedown    鼠标按钮被按下。
        2. onmouseup    鼠标按键被松开。
        3. onmousemove    鼠标被移动。
        4. onmouseover    鼠标移到某元素之上。
        5. onmouseout    鼠标从某元素移开。
        
5. 键盘事件：
        1. onkeydown    某个键盘按键被按下。    
        2. onkeyup        某个键盘按键被松开。
        3. onkeypress    某个键盘按键被按下并松开。

6. 改变事件
        1. onchange    域的内容被改变。

7. 表单事件：
        1. onsubmit    提交按钮被点击。

// 使用
1. 命名函数绑定
    1). 定义一个命名函数;
    2). 给标签添加属性:on事件名 = "函数调用";
    效果: 事件被触发的时候,函数就会被调用
2. 匿名函数绑定❤❤❤
    1). 用js代码找到标签对象;
    2). 标签对象.on事件名 = 匿名函数;
    效果: 事件被触发的时候,函数就会被调用
    匿名函数更加常用,更加解耦 



    <input type="button" value="普通函数" onclick="method01('hehe')"> <br>
    <input type="button" value="匿名函数" id="btn"> <br>
        
        function method01(a) {
        console.log(a)
    }

    //DOM: 这个方法可以看成id选择器
        // btn表示 id选择器 定位到的 标签对象

        console.log("xixi")

8、内置对象

1、String

1. 构造字符串可以使用
    a. 双引号
    b. 单引号
    c. 反引号(重音符) 键盘左上角 esc下面 (es6模板字符串)
       优势:在字符串拼接时，可直接使用${变量}引入外部变量值
    
2. 字符串常用方法
    substring(startIndex,endIndex) 提取字符串中两个指定的索引号之间的字符(包头不包尾)
    toLowerCase() 把字符串转换为小写。
    toUpperCase() 把字符串转换为大写。
    split("指定字符切割") 把字符串分割为字符串数组。
    trim() 去掉首尾空格

2、Array数组对象

1. let array = new Array('a','b','c')
2. let array = ['a','b','c']  // 推荐, 中括号
注意事项：
    a.数组中的元素可以不是同一种类型
    b.数组可以动态扩容

3、Date日期对象

1. 创建对象
   let date = new Date(); 获取当前时间时间 //Sun May 24 2020 14:38:34 GMT+0800 (中国标准时间:东八区)
2. 常用方法
    toLocaleString() 转换为本地日期格式的字符串  //2021/8/28下午9:00:06
    getFullYear() 获取日期中的year // 2021
    getMonth()   获取日期中的month // 值：0-11
    getDate()    获取日期中的号 // 24
    getTime()   获取时间毫秒值(时间原点: 1970年1月1号0时分秒) ;//// 1970年~今毫秒值

4、Math数学对象

1. 四舍五入 round()
2. 向下取整 floor()  地板
3. 向上取整 ceil()  天花板
4. 产生随机数 random() ：返回 [0,1) 之间的随机数(包含0不包含1)

5、js全局函数

js全局函数不需要通过任何对象间接调用，可直接调用;
常见的全局函数如下：
parseInt(str)：输入数据转int类型
parseFloat(str)：输入数据转小数类型
encodeURI(word)：编码字符串
decodeURI(result2)：解码字符串
